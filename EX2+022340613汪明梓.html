数据可视化练习项目

Matplotlib数据可视化综合练习

项目概述

本项目包含五个独立的数据可视化图表，涵盖网格线设置、参考线添加、饼图制作、区域突出显示和注释文本等关键功能。

技术栈： Python + Matplotlib + NumPy + Plotly

文件格式： HTML + Jupyter Notebook

1. 正弦余弦曲线图的刻度与网格设置

对应章节： 3.2.2 & 3.4.1节

实现功能：

  • 生成x范围[-π, π]的正弦sin(x)和余弦cos(x)曲线数据
  • 显示完整的网格线（x轴和y轴），灰色虚线
  • 设置y轴刻度标签为{-1,-1/2, 0, 1/2, 1}，使用LaTeX格式

2. 电影票房柱状图添加参考线和注释

对应章节： 3.5.2节

实现功能：

  • 使用2019年内地电影票房排行榜数据
  • 计算平均票房并添加垂直参考线（点虚线）
  • 为票房最高的电影添加指向型注释"年度票房冠军"

3. 果酱面包配料比例的可视化

对应章节： 3.3.2 & 3.7.1节

实现功能：

  • 饼图显示配料比例，包含百分比（%3.1f%%）
  • 添加无指向型注释文本"配料比例"
  • 图表下方显示配料重量表格
  • 爆炸效果避免百分比数字重叠

4. 三角函数正值区域突出显示

对应章节： 3.5.2节

实现功能：

  • 突出显示sin(x)和cos(x)都为正值的区域[0, π/2]
  • 浅绿色背景，透明度0.3
  • 沿用第一个图的设置和LaTeX格式刻度

5. 班级英语成绩评估图

对应章节： 3.6.2节

实现功能：

  • 高二年级各班男女生英语平均成绩柱状图
  • 添加全校平均分参考线
  • 无指向型注释文本"全校平均分"，位置(3.5,89.0)
  • 文本对齐方式：水平右对齐，垂直底部对齐
  • 浅黄色背景框，透明度0.2

交互功能说明

图表交互功能：

  • 鼠标悬停显示数据详细信息
  • 点击图例可以显示/隐藏数据系列
  • 使用鼠标可以缩放和平移图表
  • 双击可以重置图表视图

文件格式：

  • HTML文件：直接浏览器打开查看（当前文件）
  • Jupyter Notebook：在Jupyter环境中运行
  • Python脚本：生成静态图片文件
