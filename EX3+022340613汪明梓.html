1. 线型与颜色设置（4.3.2节实例2）

要求：

  • 使用2017年7月汇率数据：点划线、绿色、线宽1
  • 使用2019年7月汇率数据：长虚线、RGB(0.8,0.2,0.4)、线宽3
  • 添加图例以区分两条曲线
汇率走势图 - 图形效果模拟
// 完整代码实现 import matplotlib.pyplot as plt import numpy as np # 设置中文字体 plt.rcParams['font.sans-serif'] = ['SimHei'] plt.rcParams['axes.unicode_minus'] = False # 汇率数据 dates_2017 = [3, 8, 13, 17, 25, 31] rates_2017 = [6.8007, 6.8060, 6.7835, 6.7700, 6.7511, 6.7265] dates_2019 = [3, 8, 13, 17, 25, 31] rates_2019 = [6.8640, 6.8881, 6.8662, 6.8827, 6.8737, 6.8841] plt.figure(figsize=(10, 6)) plt.plot(dates_2017, rates_2017, linestyle='-.', color='g', linewidth=1, label='2017年7月') plt.plot(dates_2019, rates_2019, linestyle='--', color=(0.8, 0.2, 0.4), linewidth=3, label='2019年7月') plt.title('2017年7月与2019年7月美元/人民币汇率走势图') plt.xlabel('日期') plt.ylabel('汇率（人民币/美元）') plt.grid(True, linestyle=':', alpha=0.7) plt.legend() plt.tight_layout() plt.show()

2. 填充区域和数据标记（4.4.2节实例3 & 4.7.1节）

要求：

  • 绘制产品A和产品C的折线图，为两条折线添加圆形标记
  • 使用fill_between()填充两条曲线之间的区域
  • 填充颜色使用#ADD8E6（浅蓝色），透明度为0.3
产品销售额对比图 - 图形效果模拟
# 完整代码实现 quarters = ['第1季度', '第2季度', '第3季度', '第4季度'] product_a = [2144, 4617, 7674, 4409] product_c = [153, 1214, 2414, 680] plt.figure(figsize=(10, 6)) plt.plot(quarters, product_a, marker='o', linewidth=2, label='产品A') plt.plot(quarters, product_c, marker='o', linewidth=2, label='产品C') plt.fill_between(quarters, product_a, product_c, facecolor='#ADD8E6', alpha=0.3, label='A-C区域') plt.title('产品A和产品C各季度销售额对比') plt.xlabel('季度') plt.ylabel('销售额（万元）') plt.grid(True, linestyle=':', alpha=0.7) plt.legend() plt.tight_layout() plt.show()

3. 使用颜色映射表（4.2.2节）

要求：

  • 生成50个随机的x轴和y轴数据点
  • 使用scatter()绘制散点图，设置颜色参数c为随机数组
  • 颜色映射表设置为'Blues'，标记大小为100
  • 添加颜色条显示颜色映射
颜色映射散点图 - 图形效果模拟
# 完整代码实现 np.random.seed(42) x = np.random.rand(50) y = np.random.rand(50) colors = np.random.rand(50) plt.figure(figsize=(10, 6)) scatter = plt.scatter(x, y, c=colors, cmap='Blues', s=100) plt.colorbar(scatter, label='颜色值') plt.title('使用颜色映射表的散点图') plt.xlabel('X轴') plt.ylabel('Y轴') plt.grid(True, linestyle=':', alpha=0.3) plt.tight_layout() plt.show()

4. 字体设置与数据标记属性（4.5.1节实例4）

要求：

  • 最高温度曲线使用正三角标记，红边白底
  • 添加指向型注释，字体使用serif、粗体
  • 在最高温度曲线上标记最大值
温度走势图 - 图形效果模拟
# 完整代码实现 days = list(range(1, 11)) max_temps = [28, 30, 32, 31, 34, 33, 29, 31, 30, 32] min_temps = [18, 20, 22, 21, 24, 23, 19, 21, 20, 22] plt.figure(figsize=(10, 6)) plt.plot(days, max_temps, marker='^', mec='red', mfc='white', linewidth=2, label='最高温度') plt.plot(days, min_temps, marker='v', linewidth=2, label='最低温度') max_temp_day = 4 # 第5日（最大值34°C） max_temp = max_temps[max_temp_day] plt.annotate('最大气温', xy=(days[max_temp_day], max_temp), xytext=(days[max_temp_day] + 1, max_temp + 1), arrowprops=dict(arrowstyle='->', color='red'), fontfamily='serif', fontweight='bold', fontsize=12) plt.title('温度走势图') plt.xlabel('日期') plt.ylabel('温度（°C）') plt.grid(True, linestyle=':', alpha=0.7) plt.legend() plt.tight_layout() plt.show()

5. 切换主题风格（4.6节）

要求：

  • 使用plt.style.use()切换到'ggplot'主题风格
  • 绘制y=x²的曲线
  • 观察主题风格切换后的效果
ggplot主题风格折线图 - 图形效果模拟
# 完整代码实现 plt.style.use('ggplot') x = np.linspace(-5, 5, 100) y = x**2 plt.figure(figsize=(10, 6)) plt.plot(x, y, linewidth=2) plt.title('使用ggplot主题风格的折线图') plt.xlabel('X轴') plt.ylabel('Y轴（X²）') plt.tight_layout() plt.show()
